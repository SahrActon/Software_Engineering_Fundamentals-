There are three main categories of objects

1. Host objects-  are objects defined by the environment in which your code is run
   for example
    1. Console
    2. Document
    3. Node
    4. Comment

2. Core objects- are objects defined or built into the javascript language itself
   For example
   1. Math
   2. Object
   3. String
   4. Boolean
   5. Array
   6. Date
   7. Number (e.g = var biggestNum = Number.MAX_VALUE; || console.log(Number = 10); )

3. Objects defined by codes authors-
     These are objects programmed by the developer themself.
